# BabyGenderGuessAndReveal frontend (React)

This folder is the Create React App UI for the monorepo. **Setup, scripts, and deployment** are documented in the [**repository root README**](../README.md).

Usual commands from this directory:

- `npm install` — install dependencies  
- `npm start` — dev server (proxies API to `http://localhost:5000`)  
- `npm run build` — production build for Flask to serve from `../backend`
